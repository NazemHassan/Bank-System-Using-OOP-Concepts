# 💻 Bank System Project (C++)

I rebuilt my **Bank System project** using Object-Oriented Programming (OOP) after originally implementing it with structured programming.

In this version, I focused on improving the architecture and code organization by:
- Separating the logic layer from the front-end layer
- Dividing each core part of the system into dedicated files
- Writing more maintainable and scalable code

---

## 🔧 Project Features

### 🔹 Core 1: Client Management
- Show client list  
- Add new client  
- Delete client  
- Find client  
- Update client information  

### 🔹 Core 2: Transactions
- Deposit  
- Withdraw  
- View total balances  
- Transfer between accounts  
- Transfer register (history)  

### 🔹 Core 3: User Management
- List users  
- Add user  
- Delete user  
- Update user  
- Find user  

### 🔹 Core 4: Currency System
- List currencies based on exchange rate data  
- Find currency  
- Update currency  
- Currency calculator  

---
### 🔐 Authentication System  
- 🔑 Login / Logout  
- 🛡️ Password encryption  
- 👤 User permissions system   
- 🚪 Access control for screens  
- ⛔ Login attempt limit (max 3 tries)

## 🎯 What I Learned
- Applying OOP concepts in a real project  
- Structuring a project into multiple files  
- Separating concerns (logic vs UI)  
- Writing cleaner and more organized C++ code
